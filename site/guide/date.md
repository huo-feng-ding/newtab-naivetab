# 📅 日期

> 标准日期展示组件，支持自定义格式、字体和多语言

日期组件以简洁的文字形式展示当前日期，适合用作新标签页的时间信息补充。

## 🔧 基本功能

- 支持多种日期格式自定义（如 `2026年4月18日`、`April 18, 2026` 等）
- 日期语言跟随系统语言设置，中文环境下显示中文日期格式
- 实时更新

## 📋 日期格式自定义

通过设置面板可自定义日期展示格式，以下为常见格式示例：

| 格式效果 | 适用场景 |
|----------|----------|
| `2026年4月18日` | 中文日常使用 |
| `2026-04-18` | ISO 标准格式 |
| `April 18, 2026` | 英文环境 |
| `Saturday, April 18, 2026` | 含星期的完整展示 |
| `2026/04/18` | 简洁数字格式 |
| `4月18日 星期六` | 中文轻量格式 |

组件的日期格式遵循浏览器 `Intl.DateTimeFormat` 标准，支持配置：

| 配置项 | 可选值 | 说明 |
|--------|--------|------|
| 年份 | 显示 / 隐藏 | 控制是否展示年份 |
| 月份 | 数字(4) / 短名称(Apr) / 全称(April) | 月份展示形式 |
| 日期 | 数字(18) / 带前导零(18) | 日期数字格式 |
| 星期 | 显示 / 隐藏 | 是否展示星期几 |
| 星期格式 | 短(周六) / 全称(星期六) | 星期展示长度 |

## 🖼️ 展示效果示意

```
┌─────────────────────────────────┐
│                                 │
│    2026年4月18日 星期六          │  ← 自定义格式日期文本
│                                 │
│    字体 / 颜色 / 大小 / 阴影     │  ← 均可自定义
│                                 │
└─────────────────────────────────┘
```

## 🎨 自定义

- 可自定义字体、颜色、大小
- 可调整字间距
- 支持多层文字阴影效果（近距硬影 + 中距扩散 + 远距光晕）
- 浅色/深色主题独立配置
- 与其他组件一样可自由拖放布局
