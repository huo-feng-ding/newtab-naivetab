# 📝 备忘录

> 可编辑笔记或待办事项，临时记录信息，支持字数统计

备忘录组件提供一个可编辑的文本区域，用于在新标签页快速记录信息。

## 💡 使用场景

- **笔记/待办**：编辑日常笔记或待办清单，支持临时快速记录
- **个性展示**：可作为座右铭或喜欢的句子展示区域
- **临时备忘**：打开新标签页时随手记录想法

## 🖼️ 效果示意

```                                                
+-------------------------------------------+      
|                                           |      
|  今天的任务：                               |    
|  1. 完成项目报告                           |     
|  2. 回复客户邮件                           |     
|  3. 下午 3 点开会                           |    
|                                           |      
|                               128 字符    |      
|                                           |      
+-------------------------------------------+      
```                                                

## 🔄 两种运行模式

备忘录根据当前页面状态自动切换模式：

| 模式 | 触发条件 | 行为 |
|------|----------|------|
| **编辑模式** | 正常展示模式 | 可点击输入，正常编辑文本 |
| **只读模式** | 拖拽布局模式 | 不可编辑，防止拖拽时误触输入 |

切换流程：
```                                                  
normal mode -> click text area -> focused -> editable
drag mode   -> auto unfocus     -> read-only         
```                                                  

## ⚡ 功能特性

- 支持多行文本输入（textarea）
- 可开启字数统计，实时显示已输入字符数
- 编辑模式下正常输入，拖拽布局模式下自动切换为只读
- 聚焦时字数统计更明显
- 输入时自动保存内容，不会因刷新页面丢失

## ⚙️ 设置项

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 字数统计 | 开关 | 是否显示字符计数 |
| 占位文字 | 文本输入 | 未输入时显示的提示文字 |
| 字体 | 字体选择 | 文本字体 |
| 字体大小 | 滑块 | 文本字号 |
| 字体颜色 | 颜色选择器 | 文本颜色，浅色/深色独立配置 |
| 行间距 | 滑块 | 文本行间距 |
| 组件宽度 | 滑块 | 组件整体宽度 |
| 组件高度 | 滑块 | 组件整体高度 |

## 🎨 自定义

- 可自定义字体、颜色、大小、行间距
- 支持边框、阴影效果
- 毛玻璃背景质感（glassmorphism），带有内高光和顶部高光线
- 可调整组件宽度和高度
- 浅色/深色主题独立配置
