# 🌤 天气

> 实时天气信息，数据来自和风天气

天气组件展示当前城市的实时天气和未来预报信息。

## 🌡 实时天气

- 显示当前温度（支持摄氏度/华氏度切换）
- 显示体感温度
- 显示风力等级和风向
- 天气图标直观展示当前天气状况
- 异常天气时自动推送预警提示

## 📋 未来预报

- 可选展示未来几天的天气预报
- 预报信息可自定义显示天数

## ⚙️ 数据配置

- 数据来自和风天气 API
- 可自定义城市
- 可选温度单位（°C 摄氏度 / °F 华氏度）
- 可选风速单位（km/h 公里每小时 / mph 英里每小时）
- 内置 API Key，也可直接使用自己的 Key

## 🎨 自定义

- 可自定义字体、颜色、大小
- 可调整天气图标大小
- 浅色/深色主题独立配置
