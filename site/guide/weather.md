# 🌤 天气

> 实时天气信息，数据来自和风天气

天气组件展示当前城市的实时天气和未来预报信息。

## 🌡 实时天气

| 展示项 | 说明 |
|--------|------|
| 温度 | 当前气温，支持 °C / °F 切换 |
| 体感温度 | 综合风力、湿度等因素的体感温度 |
| 风力 / 风向 | 当前风力等级和风向 |
| 天气图标 | 直观展示当前天气状况（晴 / 阴 / 雨 / 雪等） |
| 天气预警 | 异常天气时自动推送预警提示（需要通知权限） |

## 📋 未来预报

- 可选展示未来几天的天气预报
- 预报信息可自定义显示天数

## ⚙️ 数据配置

| 配置项 | 说明 |
|--------|------|
| 城市 | 可自定义目标城市 |
| 温度单位 | °C 摄氏度 / °F 华氏度 |
| 风速单位 | km/h 公里每小时 / mph 英里每小时 |
| API Key | 使用内置 Key 或填入自己的和风天气 API Key |

## 🔑 使用自己的 API Key

内置的 API Key 有调用频率限制，如果频繁调用或多人共享，建议申请自己的 Key：

1. 访问 [和风天气开发者平台](https://console.qweather.com/)
2. 注册账号并创建应用
3. 获取 API Key（Subscription Key）
4. 在天气组件设置中粘贴 API Key


## 🎨 自定义

- 可自定义字体、颜色、大小
- 可调整天气图标大小
- 浅色/深色主题独立配置
