# 🎯 专注模式

> 选择只显示需要的组件，一键切换，减少干扰，让你的新标签页更加专注

在日常使用中，你可能只需要在新标签页上展示少数几个组件。专注模式可以帮助你隐藏其他组件，保持页面简洁。

## 🔧 使用方法

1. 在设置面板中进入 **专注模式** 设置
2. 勾选你希望显示的组件
3. 开启专注模式后，仅展示选中的组件

## ⚡ 快捷切换

- 通过 **鼠标右键菜单** 可快速开关专注模式
- 在右键菜单中选择 **专注模式** 即可切换
- 关闭专注模式后，所有组件恢复正常展示
- 可在设置面板中编辑当前专注模式选择的组件

## 💡 使用场景

| 场景 | 推荐组件 |
|------|---------|
| 🧘 极简模式 | 仅保留时钟，干净纯粹 |
| 💼 工作场景 | 时钟 + 备忘录，减少干扰 |
| 🚀 效率场景 | 键盘书签 + 搜索栏，快速访问常用网站 |
