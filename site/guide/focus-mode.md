# 🎯 专注模式

> 选择只显示需要的组件，一键切换，减少干扰，让你的新标签页更加专注

在日常使用中，你可能只需要在新标签页上展示少数几个组件。专注模式可以帮助你隐藏其他组件，保持页面简洁。

## 🔧 使用方法

### 方式一：右键菜单快速开关

```                                          
+------------------------------------------+ 
|  Focus Mode                               |
+------------------------------------------+ 
|                                          | 
|  [x] Toggle ON                            |
|                                          | 
|  Show Components:                         |
|  [x] Clock                                |
|  [ ] Date                                 |
|  [x] Memo                                 |
|  [ ] Weather                              |
|  [x] Search                               |
|  [ ] Bookmark Folder                      |
|  ... (all added components)               |
|                                          | 
+------------------------------------------+ 
```                                          

- 开启后，仅显示你在设置中选中的组件
- 关闭后，所有组件恢复正常展示

### 方式二：设置面板详细配置

1. 在空白处右键 → **通用设置**
2. 找到 **专注模式** 设置项
3. 勾选你希望显示的组件（可多选）
4. 开启专注模式开关

```                                                 
+--------------------------------------------------+
|  Focus Mode Settings                             |
+--------------------------------------------------+
|                                                  |
|  [x] Enable Focus Mode                           |
|                                                  |
|  Visible Components:                             |
|  [x] Clock                                       |
|  [ ] Date                                        |
|  [x] Memo                                        |
|  [ ] Weather                                     |
|  [x] Search Bar                                  |
|  [ ] Bookmark Folder                             |
|  ... (all added components listed here)          |
|                                                  |
+--------------------------------------------------+
```                                                 

## 💡 使用场景

| 场景 | 推荐组件 | 说明 |
|------|---------|------|
| 极简模式 | 仅保留时钟 | 干净纯粹，只看时间 |
| 工作场景 | 时钟 + 备忘录 | 减少干扰，专注手头任务 |
| 效率场景 | 键盘书签 + 搜索栏 | 快速访问常用网站和搜索 |
| 日程场景 | 时钟 + 日期 + 日历 | 关注时间和日程安排 |
| 生活场景 | 时钟 + 天气 + 资讯 | 出门前查看天气和新闻 |

## ⚙️ 设置项

| 设置项 | 类型 | 说明 |
|--------|------|------|
| 专注模式开关 | 开关 | 总开关，开启后仅显示选中的组件 |
| 显示组件列表 | 多选复选框 | 勾选要保留的组件 |

## 💡 常见问题

| 问题 | 解决 |
|------|------|
| 专注模式开启后某个组件不见了 | 在设置中勾选该组件，或关闭专注模式 |
| 关闭专注模式后所有组件位置乱了 | 专注模式只控制显示/隐藏，不会改变组件位置，检查是否误触拖拽 |
| 新添加的组件在专注模式下不显示 | 专注模式启用后，新组件默认不在显示列表中，需手动勾选 |
