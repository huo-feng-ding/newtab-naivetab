# 📐 编辑布局

> 将组件拖放到屏幕上的任意位置，打造专属布局

在编辑布局模式下，你可以自由添加、移动和删除组件。

## 🚪 进入方式

| 操作 | 说明 |
|------|------|
| **首次安装** | 自动进入编辑布局模式 |
| **右键菜单** | 在空白处右键 → **编辑布局** |
| **退出** | 点击左上角「退出」按钮，或按 `Esc` 键 |

## ➕ 添加组件

1. 在底部组件面板中，点击要添加的组件图标
2. 选中的组件会进入待放置状态
3. 拖动到屏幕内的目标位置，松开鼠标即可放置
4. 重复以上步骤可添加多个组件

## 🔄 移动组件

1. 按住已放置的组件
2. 拖动到新的位置
3. 松开鼠标完成移动

组件位置会自动保存，下次打开新标签页时保持不变。

## 🗑 删除组件

方式一：拖拽删除
1. 按住要删除的组件
2. 拖动到左上角的 **垃圾桶** 区域
3. 松开鼠标即可移除

方式二：键盘删除
1. 选中要删除的组件（单击）
2. 按 `Delete` 或 `Backspace` 键

## 💡 常见问题

| 问题 | 解决 |
|------|------|
| 组件拖不动 | 确认当前处于编辑布局模式，非编辑模式下组件不可拖动 |
| 组件位置没保存 | 退出编辑布局时会自动保存，确认点击了「退出」或按了 `Esc` |
| 屏幕空间不够 | 缩小浏览器窗口或使用更高分辨率，组件支持所有分辨率适配 |
| 想恢复默认布局 | 在通用设置中使用「重置设置」→ 快速重置（保留用户数据） |
