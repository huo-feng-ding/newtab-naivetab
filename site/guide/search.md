# 🔍 搜索栏

> 可自定义搜索引擎，支持自动补全建议，快速检索信息

搜索栏组件提供便捷的搜索入口，可在新标签页直接进行搜索。

## 🔧 基本功能

- 支持自定义搜索引擎，如 Bing、Google、百度等
- 可自定义引擎 URL 格式（使用 `{query}` 占位符），方便添加任意搜索引擎
- 输入时提供自动补全建议（基于百度搜索建议 API），最多显示 6 条
- 支持键盘上下键浏览建议列表，回车直接搜索
- 可选显示搜索按钮图标

## 🔗 打开方式

| 打开方式 | 说明 |
|----------|------|
| **当前标签页** | 默认方式，在当前标签页打开搜索结果 |
| **新标签页** | 在新标签页打开搜索结果，保留当前页面 |

## ⌨️ 交互操作

```                                                             
+------------------------------------------------------+        
| [Q] 搜索内容...                                      |        
+------------------------------------------------------+        
|  搜索建议 1                                          |        
|  搜索建议 2                                          |        
|  搜索建议 3  ← 当前选中                              |        
|  搜索建议 4                                          |        
|  搜索建议 5                                          |        
|  搜索建议 6                                          |        
+------------------------------------------------------+        
```                                                             

| 操作 | 行为 |
|------|------|
| 点击输入框 | 聚焦，展开建议下拉菜单 |
| 上 / 下方向键 | 上下切换选中的建议项 |
| Enter | 搜索（如有选中建议项则使用该项） |
| Escape | 取消建议选中 / 收起建议列表 |
| 点击 x | 清空输入内容 |
| 失焦 | 自动收起建议列表 |

> 拖动布局模式下禁止输入，确保拖拽操作不受干扰。

## 🌐 自定义搜索引擎

### 常用搜索引擎 URL 模板

| 搜索引擎 | URL 模板 |
|----------|----------|
| Google | `https://www.google.com/search?q={query}` |
| Bing | `https://www.bing.com/search?q={query}` |
| 百度 | `https://www.baidu.com/s?wd={query}` |
| DuckDuckGo | `https://duckduckgo.com/?q={query}` |
| 搜狗 | `https://www.sogou.com/web?query={query}` |
| 必应国际 | `https://www.bing.com/search?q={query}&cc=us` |
| 站内搜索 | `https://www.google.com/search?q=site:example.com+{query}` |

### 添加步骤

1. 在搜索栏组件上右键 → **组件设置**
2. 在 **搜索引擎** 下拉中选择「自定义」
3. 输入搜索引擎名称（如「Google」）
4. 输入 URL 模板，用 `{query}` 作为搜索词占位符
5. 保存后生效

## 🎨 自定义

- 可自定义占位文字（不设置时显示当前搜索引擎名称）
- 支持边框、阴影效果
- 毛玻璃背景质感，聚焦时背景微微提亮
- 可调整组件宽度、高度、圆角、内边距
- 浅色/深色主题独立配置
