## 🌟 功能特性

### 内置组件

| 组件 | 说明 |
|------|------|
| **⌨️ 书签键盘** | 以物理键盘形态展示书签，一键直达目标网站。支持设置全局快捷键，在浏览器任意界面快速调出，效率拉满。 |
| **📁 文件夹书签** | 以网格形式浏览原生书签文件夹，支持进入/返回子文件夹，图标与名称可选显示，列数、间距、圆角、行高均可自由调整，支持在新标签页打开链接。 |
| **🕐 时钟** | 数字时钟、模拟时钟、翻牌时钟、霓虹时钟，多种风格任你选择。 |
| **📆 日期** | 清晰展示当前日期，可自定义格式与样式。 |
| **📅 万年历** | 内含农历、节假日、法定假日信息，轻松查看工作日安排，合理规划个人日程。 |
| **📊 年度进度** | 以天数、百分比和点阵图直观呈现今年已过去的时间，感受每一天的流逝与积累。 |
| **🔍 搜索栏** | 支持自定义搜索引擎，快速发起搜索。 |
| **📝 备忘录** | 可编辑的笔记与待办事项，随时记录灵感与任务。 |
| **🌤 天气** | 实时展示温度、风力、湿度、生活指数等，异常天气自动预警，数据来自和风天气。 |
| **📰 资讯** | 聚合多种平台热榜，一屏掌握全网动态。 |

### 核心能力

- 🖱️ **自由布局** — 所有组件均可拖拽至屏幕任意位置，兼容各种屏幕分辨率
- 🎨 **深度定制** — 字体、大小、颜色、背景、边框、阴影、模糊效果，样式全面可调
- 🖼️ **背景自定义** — 支持 Bing 每日壁纸、本地图片，亦可固定单张背景
- 🌗 **自动主题切换** — 跟随系统外观自动切换浅色/深色模式，夜间使用不伤眼
- 🎯 **专注模式** — 只展示你需要的组件，右键菜单一键切换，沉浸式使用体验
- ☁️ **多设备同步** — 依托浏览器同步功能实现跨设备配置同步（需登录账号），并支持导出文件本地备份
- 🌐 **多语言** — 支持简体中文与英文
- 🔒 **完全开源，零数据收集** — 不记录任何隐私信息或操作行为，你的数据只属于你
