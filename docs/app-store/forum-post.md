# 🔷 NaiveTab — 用可视化键盘管理浏览器，我的开源新标签页扩展

NaiveTab 从 2021 年开始开发，历经四年打磨，最近推到了 2.2 版本，加入了书签键盘、指令键盘等新功能，想分享给大家试试。

项目地址：https://github.com/GXFG/newtab-naivetab

> [📷 截图：NaiveTab 新标签页全貌，展示键盘 + 时钟 + 搜索栏等组件的布局]

## 它是什么

日常浏览器书签越存越多，找网站要翻好几层文件夹；浏览器自带的快捷键难记还容易冲突。

NaiveTab 是一个开源的浏览器新标签页扩展（支持 Chrome / Edge / Firefox），核心理念是：

> **把键盘布局变成浏览器的操作面板，用肌肉记忆代替层层点击。**

它不是一张静态壁纸 + 几个书签链接，而是一个**可交互、可拖拽、全参数可调**的新标签页工作台。

---

## ⌨️ 核心亮点：可视化键盘

这是 NaiveTab 最有辨识度的功能，市面上比较少见。

### 书签键盘

以真实键盘布局展示书签，按下对应按键即可打开目标网站。

> [📷 截图：书签键盘布局，标注按键映射关系]

- 自定义按键映射 —— 哪个键开哪个网站，完全自己定
- 自定义键盘配列 —— 全尺寸 / 87 / 60% 布局可选
- 自定义键帽主题 —— 配色、字体、样式全面可调
- 书签来源可切换 —— 支持不同的书签文件夹

**使用场景：** 打开新标签页 → 扫一眼键盘就知道哪个键开 GitHub → 按下 `G` 直达，不用在书签栏里翻找。

### 指令键盘（v2.2 新增）

通过自定义修饰键组合（如 Ctrl+Shift）+ 按键，在**任意页面**执行 36 条指令：

- 标签页操作（关闭 / 静音 / 固定 / 新建 / 复制链接等）
- 标签组管理（创建 / 折叠 / 关闭组）
- 页面滚动控制
- 恢复关闭标签页
- ……

支持自定义修饰键录制、域名黑白名单，无需再记浏览器原生快捷键。

## 🌟 不只是键盘

NaiveTab 内置了丰富的 Widget 组件，全部支持**自由拖拽布局**：

| 类别 | 组件 |
|------|------|
| 🕐 时间日期 | 数字时钟、模拟时钟、翻牌时钟、霓虹时钟、日期、万年历（农历 / 节假日 / 法定假日） |
| 🔍 效率工具 | 搜索栏、文件夹书签、备忘录、年度进度、天气、资讯热榜 |
| ⏱️ 倒计时 | 自定义时长 + 进度环 + 计时通知 |

## 🎯 更多功能

- **自由拖拽布局** —— 组件随意放置，适配所有分辨率
- **全参数自定义** —— 字体、颜色、背景、边框、阴影，全面可调
- **玻璃光感设计** —— 统一的毛玻璃 UI 语言，视觉上比较干净
- **动态壁纸** —— Bing 每日壁纸 + 本地图片，随时换肤
- **自动深色模式** —— 跟随系统外观，夜间护眼
- **专注模式** —— 右键一键切换，只显示你需要的组件
- **跨设备云同步** —— 基于 Chrome Storage Sync，支持本地备份还原
- **多语言** —— 简体中文 + 英文
- **完全开源** —— GPL-3.0 协议

## 关于

一切始于我对 60% 配列键盘的喜爱——日常用的网站就那几个，但不想为了打开书签频繁在 100% 键盘上按方向键或鼠标。于是先做了一个 WEB 原型（[在线试玩](https://gxfg.github.io/newtab/)），用键盘布局展示书签，按对应的键就能一键打开。

后来不满足于只做书签页，慢慢加了拖拽布局、各种时钟风格、指令键盘、专注模式……发展成了现在完整的 NaiveTab。

如果你也喜欢用快捷键代替鼠标、想自定义新标签页的布局，或者对键盘布局感兴趣，NaiveTab 应该合你胃口。
